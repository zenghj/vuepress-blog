# JSX 的使用

在写 react 应用时我们通常会使用 JSX

<MyButton color="blue" shadowSize={2}>
  Click Me
</MyButton>

最终会通过 babel 把这段代码编译成 js 代码

React.createElement(MyButton, { color: 'blue', shadowSize: 2 }, 'Click Me')

那么这背后是如何实现转换的呢？

# Babel

babel是一个 JS 编译器，目的是将 ES6+的语法转换成 ES5 的语法
